# Client

The Manhunt PWA — **React 19 + Vite**, with `vite-plugin-pwa` (installable,
offline app shell) and a **Socket.IO** connection to the game server. This is
the scaffold: a landing shell that establishes the socket and shows its live
status. The lobby / map / game-over screens and GPS capture are tracked in the
backlog.

## Scripts

Run these from the repo root (they are root workspace scripts):

```bash
npm run dev:client   # Vite dev server on :5173, proxies /socket.io + /health to :3000
npm run build        # build the client into dist/ (repo root), which the server serves in production
npm run test:client  # Vitest component tests (jsdom) — client suite only
npm run test:e2e     # Playwright end-to-end tests
```

Root `npm test` runs both the server and client suites. From inside `client/`
you can also run the workspace's own scripts directly (`npm run dev`,
`npm run build`, `npm test`, `npm run test:e2e`).

During development the Vite dev server proxies `/socket.io` and `/health` to the
game server on `:3000`, so run `npm run dev` (the server) alongside it. Override
the proxy target with `DEV_PROXY_TARGET` (a Node-only variable — the browser
always connects same-origin, so this never leaks into the bundle). Set
`VITE_SERVER_URL` only to point the browser at a genuinely different, reachable
origin instead of proxying.

## Layout

```text
client/
├── index.html            # app entry
├── vite.config.ts        # Vite + PWA + dev proxy + Vitest config
├── playwright.config.ts   # e2e config (builds + boots the real server)
├── public/               # PWA icons (generated by scripts/gen-icons.py)
├── scripts/gen-icons.py  # regenerates the icons (requires Pillow)
├── src/
│   ├── main.tsx          # React root + service-worker registration
│   ├── App.tsx           # landing shell + socket status
│   ├── socket.ts         # Socket.IO client (same-origin by default)
│   └── test/setup.ts     # Vitest + jest-dom setup
└── e2e/                  # Playwright specs
```

## Icons

The PWA icons in `public/` are generated pixel art (red diamond + rings on the
dark theme). Don't hand-edit the PNGs — tweak `scripts/gen-icons.py` and run
`python3 scripts/gen-icons.py` (requires `pip install Pillow`).
